# Contributing

Accuracy is the point of this guide. The most valuable contribution is an issue pointing at something that is wrong or has gone stale — Cowork ships changes frequently and guides rot fast.

- **Corrections:** open an issue with a link to the official Anthropic source that contradicts the guide.
- **Workflow contributions:** PRs welcome for scheduled-task prompts or folder taxonomies you actually run, following the existing pattern (scoped inputs, dated output file, explicit boundaries).
- **Style:** plain English, Australian spelling, claims cited to official documentation where possible. No hype.
